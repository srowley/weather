# Weather

Command-line utility that prints specified NOAA weather measurements from a specified weather station

**Usage**

```shell
$ weather KDSM temp_f wind_mph
Measurement | Value
------------+------
temp_f      | 80.0 
wind_mph    | 12.7 
```
